  Technical Implementation:

  Use pseudo-elements (::before, ::after) plus one additional positioned div for the three layers:

  .conversation-room::before {
    content: '';
    position: absolute;
    top: -100%;
    left: -100%;
    width: 400%;
    height: 400%;
    background: radial-gradient(circle at center, rgba(255,255,255,0.15) 0%, rgba(211,84,0,0.08) 30%, transparent
  60%);
    animation: rotate-slow 40s linear infinite;
    pointer-events: none;
  }

  .conversation-room::after {
    content: '';
    position: absolute;
    top: -75%;
    left: -75%;
    width: 250%;
    height: 250%;
    background: radial-gradient(ellipse at 30% 40%, rgba(211,84,0,0.12) 0%, rgba(255,255,255,0.08) 40%, transparent
  70%);
    animation: rotate-medium 50s linear infinite reverse;
    pointer-events: none;
  }

  @keyframes rotate-slow {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
  }

  @keyframes rotate-medium {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
  }
